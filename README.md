# Server_Performance_Stats
https://roadmap.sh/projects/server-stats

🛠️ How to Use:
Save the script:

nano server-stats.sh
Paste the script into the file and save.

Make it executable:

chmod +x server-stats.sh
Run the script:

./server-stats.sh
